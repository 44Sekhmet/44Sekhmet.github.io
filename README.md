## 📡 關於

這個 repo 是 iOS App **《台日地震監測》** 的官方網站,使用 GitHub Pages 部署,作為 App 的對外介紹頁面與 AdMob 驗證入口。

> 整合台灣、日本地震資料,陪你掌握每一次震動。  
> 即時監測 · 歷史分析 · AI 解讀 · 防災知識。

## 📊 App 核心功能

| 模組 | 功能 |
|------|------|
| 🇯🇵 日本地震監測與速報 | 即時資訊、震度規模、發生位置 |
| 🇹🇼 台灣地震監測 | 即時監測、震央震級呈現 |
| 📈 台日歷史數據分析 | 比對兩地分布、看懂規律 |
| 🤖 AI 地震數據解讀 | 自動分析、簡單理解風險 |
| 🛡️ 地震防災知識 | 應變指南、防災準備 |
